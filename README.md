# Blackjack

A simple Blackjack game built with HTML, CSS, and JavaScript.

## Play

1. Click **Deal**
2. Hit to draw cards.
3. Stand to let the dealer play.
4. Dealer stands on 17.

## Deploy

Push the files to GitHub and enable GitHub Pages:

Settings → Pages → Deploy from Branch → main → /(root)

Your game will be live at:
https://s3yonkul.github.io/Blackjack-website
