# ♠ Blackjack ♦

A simple browser-based Blackjack game built with HTML, CSS, and JavaScript.

## 🎮 Features

- Classic Blackjack rules
- Player vs Dealer gameplay
- Hit and Stand controls
- Automatic dealer decisions
- Ace handling (1 or 11)
- Betting system
- Persistent bankroll during gameplay
- Responsive design
- Runs entirely in the browser

## 📁 Files


blackjack/

├── index.html

## 🚀 How to Run

1. Download or clone this repository.
2. Open `index.html` in your browser.
3. Place a bet.
4. Click **Deal**.
5. Try to beat the dealer without going over 21.
